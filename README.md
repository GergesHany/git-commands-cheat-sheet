# git-commands-cheat-sheet


### Install Git

Open your terminal and run these commands

```cpp
sudo apt update
sudo apt upgrade
sudo apt install git
```

## Configuration
| Command | Description |
| - | - |
| git config --global user.name `"Gerges"`              | Set user name |
| git config --global user.email `"Gerges@gmail.com"` | Set user email |
